Funput — Bộ gõ tiếng Việt cho macOS
====================================

CÀI ĐẶT
-------
1. Double-click "Install Funput.command".
   (Nếu macOS chặn vì là tệp tải về: chuột phải → Open → Open.)
2. Mở: System Settings → Keyboard → Input Sources → +  → Vietnamese → Funput.
3. Nếu chưa thấy Funput trong danh sách, hãy log out rồi log in lại một lần.

SỬ DỤNG
-------
- Chọn Funput trên menu bar (biểu tượng "FU") để bật gõ tiếng Việt.
- Hỗ trợ kiểu gõ Telex và VNI (đổi trong Settings của Funput).

GỠ CÀI ĐẶT
----------
- Xóa: ~/Library/Input Methods/Funput.app
- Bỏ chọn Funput trong System Settings → Keyboard → Input Sources.

Mã nguồn mở — https://github.com/  (xem README dự án).
